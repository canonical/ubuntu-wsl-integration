# Contributing

## Setup

1. install the dependency:
```
# apt install -y debhelper \
dh-python \
python3 \
python3-autopep8 \
python3-flake8 \
python3-urwid \
python3-distutils \
python3-setuptools \
gettext
```
2. clone the repository:
```
$ git clone https://github.com/canonical/ubuntu-wsl-integration
```
